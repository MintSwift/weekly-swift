## 기능
- 주간스위프트
    - iOS & Swift 관련 뉴스, 블로그 등 다양한 자료 제공
    - 2 ~ 10일 사이 간격 다양한 자료 업데이트
    - 개발자들에게 읽을 거리 및 튜토리얼을 제공

- 추천블로그
    - 추천 사이트 / 블로그 / 미디엄 웹페이지 수집하여 제공

- 위젯
    - 다양한 사이즈로 최신의 자료 제공

- 설정
    - 문의하기 / 추천 웹페이지 소개하기 
    - 다크모드
    - 광고 ( 생계형 광고 )
       - 광고 제거를 하지 않아도 모든 서비스 이용 가능
